SecuFleet Favicon Demo

This page demonstrates the circular SecuFleet favicon with 50% border radius.

Favicon Sizes:

16x16px

32x32px

64x64px

Usage in Next.js:

To use this favicon in your Next.js app, add these lines to your src/app/layout.js:

<link rel="icon" type="image/x-icon" href="/favicon/favicon.ico">
<link rel="icon" type="image/x-icon" sizes="32x32" href="/favicon/favicon.ico">
<link rel="icon" type="image/x-icon" sizes="16x16" href="/favicon/favicon.ico">

CSS Classes Available:

  • .favicon - 32x32px circular favicon
  • .favicon-large - 64x64px circular favicon
  • .favicon-small - 16x16px circular favicon
  • .favicon-icon - For icon usage with white background
